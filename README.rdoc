Active Accounting.

Application helps small businesses to manage their accounting stuff.
Among features:
- manage counterparties 
- keep track of finance activities
- generate Invoices
- productivity charts
- manage working hours
- many more

The system is not oriented for wide use but could be very useful for small IT-companies.
